# Daily theme images (calming visuals)

Drop one calming image per theme here, named `<key>.png` (or `.jpg`). The intro
screen of `/knowyouremotionalhealth` shows `/emotions/<today's-key>.png`. If a
file is missing, it silently falls back to the animated breathing orb — so you
can add them gradually.

Recommended: ~1200×760 (landscape), soft/warm, no text, no faces. Calm nature,
light, abstract textures work best.

IMPORTANT — keep each file UNDER ~250 KB. The hero is only ~190px tall, so large
files (1–2 MB) just slow the page with no visual benefit. Resize/compress before
adding (e.g. squoosh.app, or export at ~1200px wide, quality ~75).
